#### :thinking: Como rodar

```bash

# Clone o repositório
git clone https://github.com/MayaraRocha95/projeto-alura-studies

# Entre no diretório
cd projeto-alura-studies

# Instale as depedências
npm install

# Rode o projeto
npm start
```
